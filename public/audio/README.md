# Audio Files for Descent

Place the following audio files in this directory:

## Required Files

| File | Phase | Description |
|------|-------|-------------|
| `edge.mp3` | The Edge | Wind, distant heartbeat, anticipation |
| `plunge.mp3` | The Plunge | Rushing air, increasing wind, acceleration |
| `death.mp3` | Heat Death, Boiling, Crushing, Incineration | Distorted sounds, muffled, fading |
| `long-fall.mp3` | The Long Fall | Deep rumble, distant echoes, vast emptiness |
| `outer-core.mp3` | Outer Core | Liquid metal sounds, electromagnetic hum |
| `inner-core.mp3` | Inner Core | Crystalline resonance, harmonic overtones |
| `center.mp3` | The Center | Low harmonic drone, peace, resolution |

## Specifications

- Format: MP3 (or OGG for better browser support)
- Bitrate: 128-192 kbps
- Sample rate: 44.1 kHz
- Duration: Each track should be at least 30-60 seconds and loop seamlessly
- Volume: Normalize to -14 LUFS for consistent levels

## Mood Guide

1. **Edge** - Tense, anticipatory. Think standing at the edge of a cliff.
2. **Plunge** - Exhilarating, rushing. Free fall begins.
3. **Death** - Unsettling, distorted. The body fails.
4. **Long Fall** - Meditative, vast. Days of falling through rock.
5. **Outer Core** - Awe-inspiring, alien. Liquid iron ocean.
6. **Inner Core** - Wondrous, crystalline. Solid iron sphere.
7. **Center** - Transcendent, peaceful. Weightless at Earth's heart.

## Testing

Without audio files, the experience will still work but silently. Console warnings will indicate missing files.
